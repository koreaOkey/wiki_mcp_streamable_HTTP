# Confluence Personal Access Token 생성 가이드

## 1단계: Confluence 접속
1. 브라우저에서 http://localhost:8090 접속
2. 관리자 계정으로 로그인

## 2단계: 토큰 생성
1. 우측 상단 프로필 아이콘 클릭
2. "설정" (Settings) 또는 "프로필" 클릭
3. 좌측 메뉴에서 "Personal Access Tokens" 클릭
4. "Create token" 버튼 클릭

## 3단계: 토큰 설정
- **Token name**: wiki-mcp-token (또는 원하는 이름)
- **Expiry date**: 만료 날짜 설정 (장기 사용시 길게 설정)
- **Create** 버튼 클릭

## 4단계: 토큰 복사
⚠️ **중요**: 생성된 토큰은 한 번만 표시되므로 반드시 복사해두세요!

## 5단계: 토큰 적용
생성한 토큰을 알려주시면 다음 파일들을 업데이트하겠습니다:
- Docker 컨테이너 환경변수
- client.py 기본값

---

## 권한 확인
토큰이 다음 권한을 가지고 있는지 확인하세요:
- ✅ Read (읽기)
- ✅ Write (쓰기)
- ✅ Search (검색)

---

## 문제 해결
- 403 오류: 토큰이 만료되었거나 권한 부족
- 401 오류: 토큰이 잘못되었거나 형식 오류
